# Stitch Awesome

A sample marketing website built for Stitch Awesome — a master tailor and alterations shop in Ridgedale Center, Minnetonka, MN.

**Live demo:** https://stitch-awesome-mn.netlify.app

## About this project

A single-page, fully responsive website featuring:
- Editorial luxury design with serif typography
- Custom CSS-illustrated wedding gown and suit visuals
- Mobile-optimized navigation
- Dark-mode-proof color scheme
- No frameworks — pure HTML, CSS, and JavaScript

Built as part of a portfolio of websites for small local businesses without web presence.
